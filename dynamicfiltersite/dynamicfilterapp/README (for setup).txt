Useful Links
-------------

This document on Google docs is incredibly helpful for initially setting up your 
computer to run this project properly.

Look at Introduction to Django (tutorial) to download and install django properly.
There is also a tutorial so that you can learn how to use django. It's also
most likely a good idea to go through the whole tutorial.

Look at MySQL installation instructions in order to install MySQL. This is because
django is a platform to use python to interact with MySQL databases.

Don't forget to also look at the MySQL commands we've listed near the bottom of
the document.

Run the django project with : python2.7 manage.py runserver
Access website at : http://127.0.0.1:8000/dynamicfilterapp/ for user's perspective
					http://127.0.0.1:8000/admin/ for admin's perspective
